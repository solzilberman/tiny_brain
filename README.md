# Basic FeedForward Neural Network
___

## Description
This is a simple nn that learns binary operators (XOR, AND, NAND).

### Example Output For XOR
`
[info] training started
[info] training completed
0, 0 -> 0.00149514
1, 0 -> 0.978651
0, 1 -> 0.976338
1, 1 -> 0.0308541
`

## Compilation
`make all`
